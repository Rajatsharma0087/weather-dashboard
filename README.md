# 🌤️ Weather Dashboard

Real-time weather application built with **React.js** and **OpenWeather API**.

👉 **[https://weather-dashboard-three-virid.vercel.app/](https://weather-dashboard-three-virid.vercel.app/)**

## ✨ Features

- 🔍 **City Search** - Search weather for any city worldwide
- 🌡️ **Real-time Data** - Current temperature and "feels like" temperature
- 💨 **Detailed Info** - Wind speed, humidity, atmospheric pressure
- 🌅 **Sun Times** - Sunrise and sunset information
- 📊 **5-Day Forecast** - Extended weather predictions
- 🎨 **Dynamic Backgrounds** - Weather-condition-based gradients
- 📱 **Fully Responsive** - Works perfectly on mobile, tablet, and desktop
- ⚡ **Fast Performance** - Optimized loading and rendering

## 🚀 Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- OpenWeather API key ([Get it free](https://openweathermap.org/api))

### Installation

```bash
# Clone repository
git clone https://github.com/Rajatsharma0087/weather-dashboard.git

# Navigate to directory
cd weather-dashboard

# Install dependencies
npm install

# Create .env file
echo "REACT_APP_WEATHER_API_KEY=your_api_key_here" > .env
echo "REACT_APP_API_BASE_URL=https://api.openweathermap.org/data/2.5" >> .env

# Start development server
npm start
